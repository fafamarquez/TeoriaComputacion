📄 Descripción
Este repositorio contiene los programas realizados para la Práctica No. 1 
de Teoría de la Computación, tenemos como objetivo implementar algoritmos que permitan
realizar operaciones sobre cadenas dentro del contexto de los lenguajes formales.

En particular, se implementaron métodos para calcular:
Prefijos de una cadena
Sufijos de una cadena
Subcadenas de una cadena
Cerradura de Kleene
Cerradura positiva

El proyecto incluye tanto los algoritmos utilizados para el cálculo como una
aplicación con interfaz gráfica, la cual permite al usuario interactuar con
el programa de forma sencilla.

⚙️ Funcionamiento del Programa
El programa está dividido en dos secciones principales:
1️⃣ Cálculo de Prefijos y Sufijos
En esta sección el usuario introduce una cadena y el programa genera:
Todos los prefijos posibles
Todos los sufijos posibles
Esto permite analizar las diferentes subcadenas que se pueden obtener a partir de una cadena original.

2️⃣ Cerraduras de un Lenguaje
En esta sección se calculan:
Cerradura de Kleene (A*)
Cerradura Positiva (A⁺)
El usuario introduce una cadena y especifica la cantidad de elementos que desea visualizar,
con el fin de observar cómo se generan las concatenaciones de la cadena dentro del lenguaje.

🖥️ Interfaz del Programa
La aplicación cuenta con una interfaz gráfica desarrollada en Java (Swing), en la cual:
Se utilizan eventos para controlar la interacción del usuario.
Los elementos de la interfaz aparecen de forma secuencial, guiando al usuario durante el proceso.
Se permite seleccionar el ejercicio que se desea realizar y posteriormente ingresar la cadena correspondiente.
Esto mejora la experiencia de uso y evita errores al introducir los datos.

🧠 Conceptos Clave
Durante el desarrollo de esta práctica se aplicaron conceptos fundamentales de la Teoría de la Computación, tales como:
Lenguajes formales
Operaciones sobre cadenas
Prefijos y sufijos
Concatenación de cadenas
Cerradura de Kleene
Cerradura positiva
